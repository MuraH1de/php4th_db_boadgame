## ①課題内容（どんな作品か）

**すごろくゲームを更新しました。**<br>
```前回よりもログインしなくても表示される画面追加、ログイン/ログアウト機能の追加、管理者用のリンク追加。```

【ファイル概要】
- ```index.php```  **（更新箇所）**<br>
    ⇒ログインしなくても表示される画面<br>
    　1人用で1マスずつしか進めない体験版になってます。
- ```login.php```  **（更新箇所）**<br>
    ⇒ログイン画面になります。（見栄えを整えます）
- ```start.php```  <br>
    ⇒すごろくボードの設定を行う<br>
    　edit_boad.php  :  編集<br>
    　insert_boad.php  :  追加<br>
    　delete_boad.php  :  削除<br>
    　**一番下のリンクが管理者用と一般用で異なる仕様にしています。（更新箇所）**
- ```number_select.php```  <br>
    ⇒遊ぶ人数を選択する
- ```number_update.php```  <br>
    ⇒DBに遊ぶ人数を更新する（UPDATE）
- ```user_regist.php```  <br>
    ⇒名前を登録する
- ```user_insert.php```  <br>
    ⇒DBに名前を登録する（INSERT）
- ```game_index.php```  <br>
    ⇒サイコロを振って、何マス進むか決める<br>
    ⇒DBからゲームの進捗を取得して表示する（SELECT）
- ```game_insert.php```  <br>
    ⇒DBからゲームの進捗を取得して表示する（SELECT）<br>
    ⇒DBに進んだマスの数を更新する（UPDATE）<br>
    ⇒DBにゲームの進捗を登録する（INSERT）


【動作】
1. ```index.php``` 誰でも使える体験版の表示
2. ```login.php``` ログイン画面
3. ```start.php``` すごろくボードの設定
4. ```number_select.php``` 遊ぶ人数を登録
5. ```user_regist.php```　名前を登録
6. ```game_index.php```　ゲームを進める
